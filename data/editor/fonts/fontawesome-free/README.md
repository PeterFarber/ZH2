Font Awesome Free Solid font vendored for the editor UI.

- Upstream: https://github.com/FortAwesome/Font-Awesome
- Version: 6.7.2
- Source package: fontawesome-free-6.7.2-web.zip
- File used: webfonts/fa-solid-900.ttf
- License: see LICENSE.txt in this directory. Font files are licensed under the SIL Open Font License 1.1.

This project vendors the TrueType font because Dear ImGui can load it with its
default font atlas path without requiring a FreeType backend.
