# Runtime UI Fonts

The default UI settings point at `Inter-Regular.ttf`, which is currently the upright Inter variable TTF copied from the
official Inter 4.1 release.

The font is the Inter typeface by Rasmus Andersson, licensed under the SIL Open Font License 1.1.
Keep the `.license.txt` file next to the font and update the source note if the font is refreshed.
